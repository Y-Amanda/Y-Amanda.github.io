Java01

1.Java的历史由来

由SUN公司开发，（Stanford University Network）斯坦福联合网络学院。该学院研发研究一个系统“Solaris+Sparc”，Solaris是一个开源的Unix系统。

2.语言的进化

机器语言–汇编语言–高级语言–面向对象

3.Java之父：James Gosling 领导开发。

4.Java诞生记

91年Green计划——94年Oak语言更名java——95年发布JAVA
LOGO：Java咖啡杯，Duke企鹅

5.Java是SUN公司倡导的JamesGosling团队发明的面向对象的可跨平台的编程语言。

6.平台

CPU指令集：就是CPU中用来计算和控制计算机系统的一套指令集合，每种cpu都有其特定的指令集
Intel（CISC复杂指令集）
AMD（CISC简明指令集）
PowerPC（RISC使用精简指令集）

7.平台

操作系统：是充当用户和计算机之间交互的界面软件，不同操作系统支持不同的CPU（CPU指令集）。
三种主流操作系统（Liunx，Windows，Mac）都支持CISC指令集。

8.语言执行方式的三种分类：

编译执行：缺点（无法跨平台），优点（速度快）
解释执行（转译执行）：html，JavaScript
中间码+虚拟机

9.JDK

开发Java需要从需要从源文件到中间中间字节码的编译器。
运行java需要Java解释器
JDK = Java编译器 + Java解释器 + 其他组件

10.JDK安装

###（1）解压后有两个文件（jdk+jre）
jdk：开发+运行
jre：运行

###（2）jdk里面的文件
scr.zip：java源码
db：Derby数据库 （小型的高性能的数据库）
demo，sample：例子
bin：java在Windows上的可执行文件（.exe文件）
include：.h文件，与Windows交互的文件
lib：jar包，Java的类库文件

11.JDK配置

JAVA_HOME : jdk的安装根目录
CLASS_PATH : .;jdk的安装根目录\bin\dt.jar;jdk的安装根目录\bin\tools.jar
PATH:jdk的安装根目录\bin;

配置好，在cmd中运行指令javac，java，javadoc

12.三个概念说明

JVM：Java虚拟机，包含：类加载器，字节码校验器，Java解释器
JRE：Java运行时环境，包含JVM与Java运行支持类库
Java SDK：Java Software develop kit，JDK的另一个称呼。

13.Java三个分支

JavaEE：企业级，做Java Web程序和企业级部署EJB，解决浏览器端的程序
JavaSE：客户端程序
JavaME：移动终端，塞班

14.Java语言优缺点

优点：平台无关，面向对象，高性能，多线程开发，健壮、安全（异常处理机制），简单，分布式应用，无指针，自动收集内存（垃圾回收机制）。
缺点：运行速度慢，占用资源多，复杂。

1.IDE——集成开发环境

Integrated Development Environment
包括代码编辑器、编译器、调试器和图形用户界面工具。
集成了代码编写功能、分析功能、编译功能、调试功能等一体化的开发软件套。

2. 常用JavaIDE工具

Eclipse：
Jbuilder：（趋向于淘汰）——Borland公司开发
NetBeans：SUN公司研究开发的

3. Eclipse简介

出自IBM的优秀IDE工具（集成化编程环境）
开源的、第一个提出可扩展体系结构。
灵活的插件化机制使其成为超级IDE工具，支持多种工具，如：JDT，CDT，JUniT等。
强大的提示功能，和方便的调试功能使开发效率显著提高。

4. 工作空间

运行Eclipse时基本单位。
Eclipse配置信息基于工作空间。
%eclipseHome%\configuration\.settings设置工作空间基本信息。

5. Java程序块的组成：数据类型，变量，运算符，控制流程

数据类型：确定要存储在内存中的数据的类型。
变量：存储数据的基本单元。
运算符：一个符号，用于操作一个或多个参数以得出结果。
程序时按顺序执行，控制流语句允许改变此顺序

6. 变量：

其值可以改变的量，用来放最后的值。它是一片连续的内存空间。使用的时候，它是一个存放数据的容器。

7. 变量的组成：

字母，数字，下划线组成，以字母或者下划线开头的字符串。
首字母小写，驼峰，大小写敏感。

8. 变量的使用

先声明，在赋值，再使用。
C语言变量必须先声明才能使用，而Java可以在任意地方声明变量。
C语言中变量先用局部的，而在Java中，这样不能使用，会报错：变量重复定义。
